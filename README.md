# Straddie Digital Twin Builders

A Strange But True family static site for plain-English `.md` builder questionnaires that describe L0 private spaces, L1 place meshes, L2 living boundaries and civic simulation briefs.

Public site target:

- https://auraofintelligence.github.io/straddie-digital-twin-builders/

Source repo target:

- https://github.com/auraofintelligence/straddie-digital-twin-builders

## Pages

- `index.html` - gateway
- `architecture.html` - L0/L1/L2 architecture
- `l0.html` - private space notes
- `l1.html` - place mesh notes
- `l2.html` - Moreton Bay and living boundary context
- `builders.html` - working Markdown questionnaire
- `network.html` - related Strange But True family sites
- `boundaries.html` - public/private and review guardrails
- `sources.html` - source trail

## Boundary

This is a draft public doorway and builder pattern. It is not a finished digital twin, property claim, clinical tool, government plan, cultural permission, funding commitment or surveillance system.
