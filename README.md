# Straddie Digital Twin Builders

A Strange But True family static site for plain-English `.md` prompt-builder questionnaires that help visual generators, 3D world builders and simulation agents generate accurate customisable scenes from Level 0 (L0) private spaces through Level 1 (L1) places to Level 2 (L2) living boundaries.

Public site target:

- https://auraofintelligence.github.io/straddie-digital-twin-builders/

Source repo target:

- https://github.com/auraofintelligence/straddie-digital-twin-builders

## Pages

- `index.html` - gateway
- `architecture.html` - Level 0 (L0), Level 1 (L1), Level 2 (L2) prompt architecture
- `l0.html` - private space prompts
- `l1.html` - place mesh and venue world prompts
- `l2.html` - Moreton Bay and living boundary simulation prompts
- `builders.html` - working Markdown prompt questionnaire
- `network.html` - related Strange But True family sites
- `boundaries.html` - public/private and review guardrails
- `sources.html` - source trail

## Boundary

This is a draft public doorway and prompt-builder pattern. It is not a finished digital twin, property claim, clinical tool, government plan, cultural permission, funding commitment or surveillance system.
