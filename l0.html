Level 0 (L0) private space

The first generated scene belongs to the person or place steward.

Level 0 (L0) is the home cell: a room, desk, studio, garden bed, capsule, phone, local server, private object or support space. It is where the prompt learns to respect the owner before it tries to help anyone else.

Questionnaire spine

Plain questions for Level 0 (L0) prompts.

These are the questions the builder asks before any mesh, public dashboard, world builder or AI agent gets involved.

What is the space?

Name the room, bench, garden, capsule, device cluster or private place in ordinary words.

What should it help generate?

A render, object mock-up, room layout, capsule interior, repair bench, comfort scene, safety rehearsal or private simulation.

What stays private?

Health context, identity, routines, household details, raw sensor streams, family notes and location-sensitive material.

Who may review it?

The owner, carer, clinician, venue steward, trusted friend, technician, creative lead or human reviewer.

What can be shown later?

A minimal public-safe render, anonymised object view, layout idea, maintenance need, resource request or abstracted insight.

How can it be corrected?

Change the camera, remove private details, adjust scale, delete the render, update source date or sever the sync path.

Aura boundary

Clothing, not skin.

The Aura security notes and capsule compute pages both point to the same rule: a digital twin should remain an external support layer. It can help render, remember, summarise and simulate, but it should be severable, reviewable and owned by the person it represents.
